A lightweight and flexible WordPress gallery plugin, now with Elementor support and 11 gallery types.

== Description ==

**Mini Gallery** is a simple, fast, and versatile gallery carousel plugin, now integrated with Elementor page builder!

**Features:**
- Upload and manage images easily for your marketing and content teams.
- Lightweight gallery carousel with shortcode support.
- Elementor widget support ‚Äî easily add galleries directly inside Elementor pages.
- Now includes 11 gallery types for maximum flexibility and design options.

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the provided shortcodes or Elementor widget to insert galleries into your pages.
