# ADMIN Folder Structure

```lua
/admin
    /assets
        class-mgwpp-admin-assets.php      // Handle asset enqueuing
    /dashboard
        class-mgwpp-dashboard.php          // Dashboard related logic (stats, page rendering)
    /menu
        class-mgwpp-admin-menu.php         // Admin menu registration
    /modules
        class-mgwpp-gallery-modules.php    // Gallery module logic (installed modules)
    /storage
        class-mgwpp-storage.php            // Storage related functions (data handling)
    /pages
        class-mgwpp-pages.php              // Page rendering functions (albums, galleries, security, etc.)
    /helpers
        class-mgwpp-utils.php              // Helper utilities like card rendering, toggle, etc.

```
